Overview

Project Summary: This React.js school app was built using vite for fast setup for development. It features reusable components, responsive navigation, and integration with a secure backend built on Express.js and MongoDB.
Role: Full Stack Developer

Tech Stack:

  • Frontend: React.js, Vite, React Hooks, Axios for API integration.
  • Backend: Node.js, Express.js, MongoDB, Mongoose, Restful APIs.
  • Security & Utilities: bcrypt, cors, dotenv, helmet, jsonwebtoken, multer, nodemon.

    • Deployment Details:

  • The app was deployed on Netlify as a personal project.
  • Backend Deployment Limitation: The backend could not be deployed, but the app's full functionality can be demonstrated during a presentation if required.

    • Frontend Architecture

    Component Structure:

    - Built reusable components for maintainability and scalability.
    - Utilized a hierarchal routing structure.
  • stack.jsx for routing throughout the app.
  • stack.jsx referenced in app.jsx
  • app.jsx connected to main.jsx, which was finally linked in index.html

    • Responsive Design:

    - Implemented responsive navigation using React Hooks.

    Backend Development

    Server & API Architecture

    - Built with Express.js and Node.js for handling server requests.
    - Developed RESTFUL APIs for:
  • User Authentication: Registration and Login.
  • Data Management: Fetching and updating data in MongoDB

    • Security Features:

  • bcrypt for password hashing.
  • jsonwebtoken for authentication and authorization.
  • helmet for securing HTTP headers.
  • cors for cross-origin resource sharing.

    • Data Integration & Management

    API Integration:

  • Integrated RESTFUL APIs using Axios for asynchronous request.
  • Managed data flow using useState and useEffect React Hooks.

    • Example:

  • Registration and Login data flow.
  • Fetching and displaying student data from MonogoDB

    • Challenges & Solutions

    Key Challenges:

  • Ensuring efficient state management while fetching data.
  • Securing user authentication and authorization.
  • Handling responsive navigation logic with React Hooks.

    • Solutions & Lessons Learned:

  • Implemented useState and useEffect for Optimized state management.
  • Utilized jsonwebtoken and bcrypt for robust authentication.
  • Enhanced user experience with responsive and interactive UI components.

    • Deployment & Presentation

    Deployment Details:

  • The app was deployed on Netlify as a personal project.
  • Due to backend hosting limitations, the backend was not deployed.
  • Presentation Availability: The app's full functionality can be demonstrated during a presentation if required.
